Before saving the PDF

The browser print dialog needs three settings or the saved PDF will look wrong. Set these in the dialog that's about to appear:

  1. Destination → Save as PDF
  2. Margins → None (this is the most common cause of cropped or weirdly spaced output)
  3. Headers and footers → off
Print Preview
Click Save as PDF → in the print dialog, set Destination to "Save as PDF" and Margins to "None". Headers and footers should be off.
